Exemplos do ToonTalk traduzidos para Java

O ToonTalk agora tem um tradutor para Java. Tudo que você construir no ToonTalk pode ser automaticamente convertido para Java e rodado em um navegador da Web. Aqui há alguns exemplos que foram traduzidos:

  • Seqüência Fibonacci. Este exemplo calcula uma seqüência de números em que cada número é a soma dos dois números prévios.
  • Fatorial. Este exemplo calcula o fatorial de um número.
  • Gerador de sentenças aleatórias. Este exemplo gera sentenças gramaticais, mas tipicamente simples.
  • Jogo de pingue-pongue. Este exemplo roda um jogo de pingue-pongue simples.
  • Filmes de caça. Este exemplo mostra o resultado de combinações de comportamentos construídos no ToonTalk. Os objetos saltam muros, saltam entre eles, têm objetivos, caçam-se mutuamente, entre
    outras coisas.

Por favor, relate quaisquer problemas.

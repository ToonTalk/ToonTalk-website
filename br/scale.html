As balanças comparam coisas

Dentro do Porta-Treco você encontrará uma grande pilha de balanças. Você pode usar balanças para descobrir que número é maior ou que palavra ou sentença vem primeiro em ordem alfabética. Coloque uma balança em uma abertura de uma caixa com aberturas em ambos os lados. Se você colocar números nessas aberturas, a balança irá se inclinar para o número maior (ou mais pesado). Se você a colocar, ao invés, em grupos de texto, a balança irá se inclinar para aquele que vem depois na ordem alfabética.

Dica avançada. Você pode treinar um robô para fazer alguma coisa se um número é maior que outro, e outro robô para fazer algo mais se for menor que algo (e ainda um terceiro robô para desempenhar a tarefa se os números forem iguais). O exemplo de "Troca" mostra um exemplo de treinamento de um robô para trocar dois números se o primeiro é menor do que o segundo. Observe que uma balança irá balançar e ir para frente se estiver próxima a uma abertura vazia. Portanto é importante utilizar Asper para apagar, não esvaziar, os grupos próximos à balança quando deseja fazer seus robôs trabalharem em mais tipos de caixas.

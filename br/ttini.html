Personalizações de instalação do ToonTalk

SOMENTE PARA USUÁRIOS AVANÇADOS

Você pode editar o arquivo toontalk.ini que está localizado no diretório Windows do seu computador.

  • GenerateLogs. Gera arquivos chamados log<número>.dmo no diretório ToonTalk que podem ser executados novamente. Ideal para reproduzir problemas. O valor representa o número de logs obtidos.
  • AutoexemploMaxIdle. O valor é o número de segundos que se deve esperar antes de reiniciar os exemplos quando estiver executando -autoexemplo 1 ou 2.
  • Velocidade das Legendas. O valor é a porcentagem da velocidade normal. Se for maior que 100, as legendas serão mais rápidas. Se o valor é igual a 0, o ToonTalk não exibirá legendas para os exemplos.
  • NarrationFileSuffix. O valor precisa ser SPK para narração em inglês ou TAL para narração em sueco. Os padrões para o valor apropriado da opção -language.
  • Sufixo das Legendas. O valor precisa ser UST para subtítulos com inglês americano, UKT para legendas com inglês britânico ou SET para narração em sueco. É possível ter subtítulos em um idioma diferente para a narração.
  • TextToSpeechMode. Este recurso é útil apenas se você possui instalado um dispositivo de texto para voz em seu sistema e deseja utilizar esse dispositivo em vez daquele que a versão de 32 bits do ToonTalk utiliza. Basta digitar a expressão aqui.
  • JoystickDeadZone. Este recurso funciona apenas se você estiver executando a versão de 32 bits do ToonTalk e possui um joystick ou game controller conectado ao seu computador. Um joystick às vezes não é movido, mas ele informa ao computador que ele foi movido em uma pequena área. Este recurso informa ao ToonTalk quão pequenos podem ser os movimentos para serem ignorados. O valor varia de 0 a 10.000. O padrão é 2.500.
  • MaximumNumberOfHoles. Quadros, quando eles possuem muitos furos (mais de mil), começam a diminuir o ritmo do ToonTalk. Use esta opção para alterar o padrão de 2.048 furos máximos em um quadro.
  • TurnOffJoystickAutoCenter. Este recurso funciona somente se você possui um joystick force feedback. O Autocenter utiliza os motores do joystick para simular a elasticidade de um joystick tradicional. Desativá-lo pode tornar os efeitos melhores, mas deixa o joystick mais difícil de controlar.
  • DefaultUser. Este recurso foi desenvolvido para salas de aula ou museus, onde costuma-se alterar o livro de registro, o layout e/ou a cidade. Basta personalizar esses elementos de acordo com cada nome de usuário e, em seguida, configurar o nome de usuário aqui.
  • [ButtonKeyboardEquivalents]. Permite configurar as ações de um joystick de três botões (e também o botão do meio de um mouse, que é o mesmo que o terceiro botão de um joystick). Você pode definir teclas especiais colocando-as entre os sinais de maior que e menor que "<>". As teclas especiais disponíveis são <F1>, <F2>, ... <F12>, <Escape> e <space>. Por exemplo, os valores padrão são:
      3='-'
      4=<F1>
      5=<Escape>
      6=<F2>

Rodando a interface de controle de mídia do Windows da Microsoft

O Microsoft Windows suporta um conjunto de comandos para tocar sons, músicas, apresentar vídeos e outros dispositivos de mídia. Você pode enviar esses comandos para o Windows via o MCI colocando um grupo de texto com um comando nela no sensor texto-para-MCI. Por exemplo, se o grupo tem o texto "play canyon.mid" e o arquivo canyon MIDI está em seu diretório Windows, você começará a ouvir a música. Se você então colocar o grupo de texto com "close all" (feche todos) em um sensor texto-para-MCI vazio, a música parará. Muitos livros descrevem a linguagem de comando do Microsoft MCI.

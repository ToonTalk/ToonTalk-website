ToonTalk and Text to Speech

The ToonTalk CD-ROM includes and automatically installs a text-to-speech engine so that Marty can speak. It is also used by the text-to-speech sensor so you can use text-to-speech in your own ToonTalk programs.

If you would like to try ToonTalk with another speech engine or you downloaded the beta version of ToonTalk, you can download a free text-to-speech engine from Microsoft. You'll then need to select 'Set Options' next time to run ToonTalk and select the voice you wish to hear. (You won't see an option if only 1 voice is installed.) This works only in ToonTalk version 1.1 or Beta 19 or later.

Please report any problems.

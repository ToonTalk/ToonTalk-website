Utilizando seu mouse no ToonTalk

Para simplificar as coisas, normalmente, todos os botões do mouse significam a mesma coisa no ToonTalk. Você pode configurar os aceleradores para os botões do meio e direito. O botão direito pode ser configurado de tal forma que o pressionando enquanto se segura uma ferramenta fará "utilizá-la". Em outras palavras, ele age como a "barra de espaço". O botão direito também pode ser usado para trocar textos, números e imagens. O botão do meio pode ser configurado para qualquer tecla. A forma mais simples de alterar estas configurações é selecionar a opção que você deseja depois de clicar em "Ajustar Opções" após iniciar o ToonTalk.

A configuração "default" para o botão do meio é "-" para voltar para páginas de trás e rodar para trás vários controles remotos. Isto pode ser alterado no seu arquivo toontalk.ini.

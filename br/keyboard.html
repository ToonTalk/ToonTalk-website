Comandos do Teclado de ToonTalk

Apenas para usuários avançados:

Você não precisa aprender os comandos de teclado que se seguem. Você pode fazer quase tudo apenas com o seu mouse. Muitos desses comandos de teclado apenas lhe fornecem meios mais rápidos de fazer as coisas.

Se você tem um joystick que tem mais de dois botões e está conectado ao seu computador, você pode utilizar esses botões extras em vez dos comandos de teclado abaixo.

Durante os títulos da abertura:

Aperte qualquer botão para ir para a próxima tela. " Esc" irá saltar as telas remanescentes.

Durante um exemplo:

Se você apertar qualquer botão, o ToonTalk fará uma pausa e exibirá as opções de sair, assumir o controle ou voltar o exemplo.

Voando, caminhando ou no chão.

Aperte as setas direcionais para se mover em qualquer das 8 direções (em alguns teclados, apenas 4 teclas direcionais estão nomeadas. As teclas chamadas Home, Page Up, Page Down e End devem funcionar para mover-se em diagonal. Você pode mantê-las pressionadas para continuar se movendo naquela direção. Você pode ainda pressionar duas teclas ao mesmo tempo para combinar duas direções).

Aperte a tecla "pause" para parar o ToonTalk e exibir a opção de sair, voltar mais tarde ou voltar ao ToonTalk.

Aperte F7 para acelerar o que quer que esteja acontecendo.

Aperte F8 para desligar a força de todos os robôse imagens em movimento na cidade toda. Aperte F8 de novo para ligá-los.

Aperte F11 para ter um instantâneo da tela. Você encontrará a imagem no diretório do ToonTalk nomeado frm1234.bmp, onde 1234 é o número do quadro de que você tirou o instantâneo.

Aperte F12 para tirar um instantâneo da tela e salvá-lo no clipboard do Windows.

Pagedown para fazer Márcio dizer a próxima coisa.

Pageup para fazer Márcio repetir a última coisa.

Voando e aterrissando:

Aperte s ou S para voar mais alto (Subir).Segurar o botão direito do mouse também deve funcionar.

Aperte x ou X para voar mais baixo (baixar). Segurar o botão esquerdo do mouse também deve funcionar.

Caminhando fora:

Aperte F1, h, ou H para levar o helicóptero para onde você está.

Caminhando numa sala:

Aperte qualquer botão para se sentar.

No chão ou numa mente de robô:

Aperte Escape (normalmente marcado Esc) para sair.

Aperte F1 para chamar Márcio ou, se ele já estiver por perto, para mandá-lo embora.

E se sua mão está vazia.

Aperte e ou E para erguer-se.

Aperte p ou P para pegar um item que está sob seu dedo.

Aperte g ou G para pegar e girar.

Aperte F9 para ocultar sua mão se ela está visível, ou exibi-la se estiver invisível. Veja também o controlador para a visibilidade de sua mão.

E se você não estiver segurando nada além de um grupo de texto.

Nota: A razão pela qual os seguintes comandos não funcionam em grupos de texto é que você precisa ser capaz de digitar qualquer texto. Você precisa utilizar os botões do mouse para girar ou colocar os grupos de texto.

Aperte g ou G para girá-lo. (se possível).

Aperte l ou L para largá-lo.

E se você estiver segurando uma ferramenta:

Aperte a barra de espaço para ativar e desativar a ferramenta (a varinha mágica se "desativa automaticamente" após cada utilização).

E se você estiver segurando Asper:

Aperte a ou A para alterar Asper, de forma que o ligará e ele aspirará as coisas (Aspirar é o "default").

Aperte i ou I para alterá-lo, de forma que o ligará e ele apagará coisas (Limpar).

Aperte d ou D para alterá-lo, de forma que o ligará e ele devolverá coisas (Devolver).

Aperte F2 para chamar Asper. Ele começará a funcionar na sua mão. Apertar F2 novamente irá acelerá-lo.

E se você está segurandoSoprana:

Aperte m ou M para alterar Soprana, de forma que a ligará e ela tornará as coisas maiores (Default).

Aperte v ou V para alterá-la, de forma que a ligará e ela tornará as coisas mais altas.

Aperte r ou R para alterá-la, de forma que a ligará e ela tornará as coisas mais largas.

Aperte n ou N para alterá-la, de forma que a ligará e ela tornará as coisas menores.

Aperte t ou T para alterá-la, de forma que a ligará e ela tornará as coisas mais estreitas.

Aperte u ou U para alterá-la, de forma que a ligará e ela tornará as coisas mais curtas.

Aperte b ou B para alterá-la, para que ela deixe as coisas em um "bom" tamanho.

Aperte F3 para chamá-la e ela saltará na sua mão. Apertar F3 novamente irá acelerá-la.

E se você estiver segurando a Varinha Mágica:

Aperte o ou O para fazer uma cópia do original do item sob a extremidade da varinha.

Aperte c ou C para fazer uma cópia de cada item.

Aperte z ou Z para fazer uma cópia da própria varinha.

Aperte F5 para fazer a varinha voar para a sua mão. Apertar F5 novamente irá acelerá-la.

E se você estiver segurando uma bomba.

Aperte qualquer tecla para explodir a bomba.

E se você estiver segurando ou apontando para uma agenda:

Aperte - ou "a barra de espaço" para voltar para a página anterior (última página, se estiver na primeira.)

Aperte + ou a barra de espaço para ir para a próxima página.

Entre com dígitos (0 a 9) para inserir um número de página.

Aperte F4 para fazer que a última agenda que você usou voe para você. Apertar F4 novamente irá acelerá-la.

E se você estiver segurando um número ou apontando para um número:

Entre com dígitos (0 a 9) para inserir um número.

Aperte Backspace para remover o último dígito.

Aperte +, -, /,*, x,%,^. &, ou = para inserir uma operação numérica.

Aperte g ou G para girar um número (funciona apenas se você estiver segurando um número).

E se você estiver segurando ou apontando para um grupo de texto:

Apenas digite para adicionar ao texto.

Aperte a barra de espaço para remover a última letra.

E se você estiver segurando ou apontando para uma imagem ou clipe de animação.:

Digite + ou Backspace para mudar para o próximo ciclo ou cor ou elemento.

Digite - ou Backspace para voltar.

Segure o botão control, mova o mouse e então clique com o botão esquerdo do mouse para rodar uma imagem. Ela começará com a mesma velocidade que sua mão tinha quando soltou a imagem.

E se você estiver segurando ou apontando para um som:

Qualquer tecla do teclado emitirá o som.

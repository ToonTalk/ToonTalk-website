Add Up Robot

 

What does it do?

It adds incoming numbers, and sends the result to a nest.

 

How do I use it?

  • Give it two numbers, and it will sum them.
  • Chain it with another robot, and it will sum the sequence that that robot generates.
     
     

How does it work?

The robot drops the incoming number on the one in so far, and copies the result over to the bird.

 

What can I use it for?

Summing up sequences, and using the result to generate new sequences.

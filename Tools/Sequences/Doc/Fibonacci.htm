Fibonacci Sequence (on a nest)

What does it do?

This tool generates the Fibonacci Number Sequence on a nest.

 

How do I use it?

Just give the robot the box. It will start generating the sequence.

Note that you can change the initial numbers to generate other Fibonacci number sequences.

 

What can I use it for?

For generating Fibonacci number sequences and exploring properties.

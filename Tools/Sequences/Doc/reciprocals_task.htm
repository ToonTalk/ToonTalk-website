Reciprocals task

Generating the reciprocals

The reciprocals sequence is:

1, 1/2, 1/3, 1/4, 1/5, ....

It is interesting because it gets smaller and smaller, forever, but never goes below 0.
But what about its running totals (or Partial sums)? What happens if we sum more and more terms of this sequence?
To investigate that, we first have to generate the sequence.

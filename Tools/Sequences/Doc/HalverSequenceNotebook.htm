Halver Sequence Notebook

 

What is it?

A notebook with 4 tasks involving the Halver sequence. The Halver sequence is what you get by repeatedly halving 1, i.e.:

1, 1/2, 1/4, 1/8, ... 

 

How do I use it?

Open the notebook and do the tasks.

 

How does it work?

The first task is an untrained robot, with an input box to use in training.
The other tasks involve connecting the robot you trained to the Add Up and Simplified Excel tools.

 

What can I use it for?

Learning about sequences.

Mass tool

What does it do?

It displays the mass (surface area) of an object.

How do I use it?

Put 'Mass Calc' on the back of the object who's mass you want to display. You can also set the mass by giving the bird a number.

How does it work?

A robot multiplies the object's width and height and divides the result by 1000 (the division is an arbitary scaling amount so that the numbers are in a good range for typical object sizes). Note the robot only runs when either the width or height sensor changes. The reverse happens when you give a number to the set mass bird.

What can I use it for?

Setting and displaying mass. You can change the mass of the object using Pumpy or giving a number to the bird. Note that there are occasional rounding errors when you set the mass.

Note: the tool has been updated to include auto translation of the instructions (English, Swedish and Portuguese available).

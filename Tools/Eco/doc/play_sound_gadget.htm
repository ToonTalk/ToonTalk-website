Teacher Guidance
 

Play sound gadget

The bell makes a sound every time it touches something. It also moves.

Test it in ToonTalk by pointing and pressing 'space'.

To use the tool to make other objects make sounds:

1. Suck out the bell

2. Flip it so that you see its behaviours

3. You probably don't need the move-behaviour, so suck that away

4. If you want the tool to make other sounds, just exchange the sound pad in the robot box

5. Place the flipped bell at the back of the object that should make sounds

6. Save in main notebook and test!

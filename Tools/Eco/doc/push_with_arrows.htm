Push with arrows

This gadget lets you make objects move more smoothly with the arrow keys, than with the "normal" move with arrows (which takes steps). It could for instance be used to make things that live under water to behave more realistically.

 

To get something to move like this gadget:

1. Suck out the little arrow-key picture

2. Flip it

3. Place it at the back of the object that should use it

4. Save in notebook and test!

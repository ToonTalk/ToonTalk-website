Installing ToonTalk on a Networked Server

It is possible to install the ToonTalk files on a networked file server and then install it on each licensed client machine in a minimal manner taking up very little disk space.

  1. Copy the entire ToonTalk folder on the CD-ROM to the server on the directory of your choice.
  2. Edit the toontallk.ini file's Directory section to refer to the ToonTalk folder on the server. You'll find it in the ToonTalk folder under My Documents.
  3. Copy the toontalk.ini file to each node (again the ToonTalk folder under My Documents).

If you are installing to a version of Windows prior to Windows XP then we advise that one installs on each node and then removes the files placed in the ToonTalk folder.

If your users use different computers then by setting the ToonTalk Users directory to a directory on the network server, ToonTalk will remember their state regardless of which computer they are using. To arrange this you'll have to edit the UserFiles entry of toontalk.ini file on the Windows directory of each local machine.

Please report any problems to support@toontalk.com.

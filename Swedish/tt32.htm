The 32-bit Version of ToonTalk

System requirements.

Windows 95 or Windows NT 4.0 or later. You must install the 16-bit version of ToonTalk first.

Installation over the net.

Simply run beta8c32.exe.

The 32-bit version requires the DirectX game extensions from Microsoft. You can get the latest version from Microsoft. Windows 98 and NT 4.0 already include DirectX.

Running the 32-bit version.

Beta8c32 will add an item to the Start -- Programs -- ToonTalk menu for running the 32-bit version. It will also give you the option of making it the default.

Differences between 16-bit and 32-bit versions.

  • 32-bit version mixes sounds.
  • 32-bit version can run 640x480 (but it requires a 30 megabyte file that is not available over the net).
  • Accepts long file names.
  • Marty can talk! But you need to provide a text-to-speech engine that is compatible with Microsoft's Speech API. You can select the mode by editing "TextToSpeechMode=" in your toontalk.ini file.
  • Can import larger (greater than 64K) pictures.
  • Only the 32-bit version can use joysticks and game controllers. (But not on Windows NT 4.0.)
  • Faster on high-end machines and video boards (and slower on old machines).

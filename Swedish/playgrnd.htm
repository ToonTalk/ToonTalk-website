Öppna Playgroundspel i ToonTalk

Playground filerna har uppdaterats för ToonTalk 3 (3.188 eller Beta 52) och senare. ToonTalks startdialog har en knapp för att komma åt Playgroundfilerna. Skulle de inte vara installerade kommer de laddas hem första gången de används. Du kan också lägga till filerna här.

The Playground Project var uppbyggt kring ToonTalk (och Logo) för att hjälpa små barn göra egna dataspel. The Game Place heter stället där spel som barn och forskare gjort finns samlade. Dessa spel kräver en relativt ny version av ToonTalk. Spelen har sina egna bilder och ljud. Du kan köra dataspelen och ändra dem som du vill. Du kan kan också lägga upp egna spel på The Game Place. De flesta spelen kräver ToonTalk version 2.28 elle Beta 25 eller senare.

Det normala är att dessa spel har sparats i avstängt läge. Tryck på mellanslagstangenten medan du pekar på spelet eller håller i det för att sätta igång dem. Sen kan du se om det finns saker i spelet som du vill ändra på. På baksidan, om du vänder på spelet, kan du ofta få mera information.

Med ToonTalk version 2.35 eller Beta 27 eller senare så när du klickar på länkar till spel mm på the Game Place då laddar ToonTalk hem de bilder och ljud som behövs. Om du får felmeddelanden om filer som saknas kan du pröva detta.

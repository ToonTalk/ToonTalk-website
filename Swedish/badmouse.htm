Dealing with troubles using your mouse in ToonTalk

When ToonTalk tries to find out how much your computer's mouse has moved, it sometimes finds out it has moved some impossible amount. This has been observed when running ToonTalk on an emulator like SoftWindows or old versions of VirtualPC on a Macintosh computer. Note that ToonTalk now runs without any mouse problems on VirtualPC 4.0 or higher. You need to following the following advice from Connectix:

As of Virtual PC 4.0, you can now disable host cursor integration while you're running Virtual PC. The option can be found in the Mouse Settings panel. You can turn it off temporarily while you're using a game that would otherwise operate incorrectly.

On a PC this shouldn't happen, but if it happens only when moving the mouse wildly, then you may wish to reduce the sensitivity of the mouse.

You can use ToonTalk without moving the mouse. The arrow keys on the keyboard can be used instead.

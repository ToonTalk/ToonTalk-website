Dealing with ToonTalk installation problems

You probably got to this page because something is wrong with the installation of ToonTalk on your machine. If you tried to copy files from another installation without running the installation program, then problems are common. You'll need to install ToonTalk over again using the CD-ROM if you have one or the file that was downloaded off the Internet.

Installation can fail if you have too little disk space. Normally you'll get a warning stating you have too little free disk space during installation but not always.

If you installed ToonTalk from a file you downloaded off the Internet, then check the size is correct.

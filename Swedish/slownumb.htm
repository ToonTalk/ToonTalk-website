ToonTalk kan hänga sig vid arbete med mycket stora tal

Vill du se den senaste versionen av denna sida på webben kan du klicka här.

Från version 2.50 (eller beta 34) kan ToonTalk arbeta med hur stora tal som helst (som datorn klarar av). När talen är på väg att bli väldigt stora kanske du märker att ToonTalk hänger sig ett tag med siffrorna byts ut, kopieras eller när du ska sätta dig i ett hus med väldigt stora tal på golvet. Tal med färre än 1000 siffror är inte något problem. Inte heller bör du få problem med robotar som arbetar med tal som har en miljon siffror under förutsättning att de finns i ett annat hus än det du är i eller om de ligger på baksidan av en bild. Om du däremot första gången ska se ett tal med en miljon siffror kan du få vänta i flera timmar medan ToonTalk förebereder talet för visning på skärmen.

Vill du väldigt gärna arbeta med tal som har tusentals eller miljontals siffror kan du unvika dessa långa pauser genom att se talen efter lämplig potensupphöjning. Klicka här för att lära dig hur man ändrar talens potens.

Detta problem kan också uppträda när det gäller exakta bråktal eftersom de kan innehålla väldigt stora tal, ett för nämnaren och ett för täljaren.

ToonTalk exempel på konvertering till Java

ToonTalk har en konverterare som översätter till Java. Allt du bygger i ToonTalk kan automatiskt konverteras till Java och köras i vilken webbläsare som helst. Här har du några exempel och du behöver vara uppkopplad för att kunna se dem:

  • Pingpongspel. Här kan du se ett pingpongspel i Java.

    Resten av exemplen beskrivs på engelska.

    Fibonacci sequence. This example computes a sequence of numbers where each number is the sum of the two previous numbers.

  • Factorial. This example computes the factorial of a number.

  • Random sentence generator. This example generates grammatical, but typically silly, sentences.

  • Chase Movies. This example shows the result of combining behaviors constructed in ToonTalk. The objects bounce off walls, each other, have goals, chase each other, and more.

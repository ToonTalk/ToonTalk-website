Customizing ToonTalk's New HTML Dialogs

Från och med version 2.52 eller beta 35 visar ToonTalk alla dialoger som baserade på HTML-filer som användarna kan komma åt och ändras. De dialoger som du kan ändra finns i denna lista:

  • starttt.htm Detta är den viktigaste dialogen. It is the one you see when you start up ToonTalk. It has buttons for launching ToonTalk in Free Play or Play Game modes. Den kan köra alla inbyggda demofilmer. Man kan ändra inställningar och hjälp. Genom att ända denna fil kan du bestämma vilka demofilmer och uppdrag som är tillgängliga, gömma vissa möjligheter i ToonTalk och lägga till nya möjligheter som att komma till en speciell stad som byggts i förväg upp på ett speciellt sätt. För att kunna göra dessa ändringar behöver du kunna koda i HTML och kanske göra ändringar i de JavaScript som används på dessa sidor.
  • askname.htm Den här dialogen ber användaren skriva in sitt namn och välja utseende.
  • genpause.htm Den här dialogen visas när användaren gör en paus eller avslutar ToonTalk.
  • nosave.htm Den här dialogen skiljer sig från genpause.htm eftersom den saknar möjligheten att spara staden.
  • dempause.htm Den här dialogen visas när en demofilm blivit stoppad. (Gäller enbart demofilmer som gjorts i beta 35 eller version 2.52 eller senare.)
  • messbox.htm Den här dialogen är den generella dialogen för att anslå meddelanden och för att ställa enkla Ja/Nej-frågor eller frågor som ska bekräftas med OK,.
  • clickme2.htm Den här dialogen körs när CD-ROM sätts in i CD-enheten om ToonTalk har installerats. clickme.htmligger på CD-ROM och körs om ToonTalk inte har installerats.

JavaScript används i alla HTML dialogsidorna. Informationen kommer till sidan via window.dialogArguments. detta är en rad med semikolon (';') som skiljer olika värden åt. För att få retur på informationen till ToonTalk behöver de skicka en rad tillbaka som värdet i window.returnValue och sen behöver du avsluta med window.close(). Den rad du skickar tillbaka till ToonTalk måste vara en korrekt kommandorad.

Lägg märke till att om du skapar en speciell dialogsida och ger den namnet vadsomhelst.html.tt eller vadsomhelst.htm.tt ( där "vadsomhelst" kan var vilket användbartnamn som helst) då kommer Start ToonTalk starta med den sidan om man dubbelklickar på filen.

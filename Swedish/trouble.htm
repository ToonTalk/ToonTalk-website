Vad ska jag göra om ToonTalk inte fungerar riktigt

Om du anser att ToonTalk inte fungerar riktigt, var vänlig klicka på en av rubrikerna nedan. Om problemet kvarstår när du provat de förslag på lösningar som ges där kan du rapportera felet.

  • ToonTalk stannar eller kraschar.
  • ToonTalk arbetar för långsamt.
  • ToonTalk har problem att köra demon.
  • ToonTalk installerades inte riktigt.
  • ToonTalk verkar ha problem med Microsofts DirectX speltillägg.
  • 16 bitars versionen har problem att arbeta i fullskärmsläge.
  • ToonTalk har problem att använda min joystick eller game pad.
  • ToonTalk har problem att skapa Java applets.
  • ToonTalk kan inte visa bilder i andra format än BMP.

Att rapportera om problem till ToonTalks Support

Bästa sättet är att skicka ett mejl till support@toontalk.com.

Tänk på att ge följande info:

  1. Ditt namn och e-postadress.
  2. Beskriv problemet så noga du kan.
  3. Ange vilken version av ToonTalk du använder. Klicka på 'Om'-knappen när du startat ToonTalk för att se versionens nr.
  4. Finns dxinfo.exe eller dxdiag.exe på din dator, kör den i så fall och skicka med en kopia av informationen som du får. Annars kan du beskriva vad för dator du har och vilket operativsystem du kör på. Exempelvis Pentium 4 med Windows XP Home Edition.
  5. Det är alltid bra om du kan tala om hur man kan göra så att felet uppstår igen. Eventuellt kan det vara bra att spara ToonTalk-saker som krånglar och bifoga dem.

Tack!

You need a browser that can run Java to see this. The latest browsers from Microsoft or Netscape should work fine.

This Java applet was automatically created by ToonTalk from a picture of a ball, paddle, and score all with robots on the back.

This is a very simple Ping Pong game. This version uses the keyboard to control the paddle.

  • Press the up arrow to move up
  • Press the down arrow to move down
  • Press any other key to stop

Vad du kan göra om ToonTalk arbetar för långsamt

ToonTalks hastighet beror på många olika saker. De kan vara din dators snabbhet, hur mycket minne den har, vilken typ av ljud- och videokort som sitter i den, vilken version av Windows du kör, om dina drivrutiner för ljud och video är uppdaterade och mycket annat.

Om du har hundratals robotar som kör i ett hus eller på baksidan av bilder kan ToonTalk bli långsamt. Om du använder mycket stora tal kan sakta ner. Du kan försöka med att logga in med ett annat namn för att få bort problemet.

Högupplösta bilder är långsamma att visa. Det kan också bli stor skillnad i hastighet beroende på om ToonTalk körs i fullskärmsläge eller i ett fönster. Ibland kan det bli långsamt om du ställt in det så att Marty talar engelska. Prova med att välja olika inställningar under Välja alternativ när du startar ToonTalk.

Har du ett annat program som kör samtidigt med ToonTalk kan de tävla om datorns resurser. Prova med att avsluta andra öppna program och kör enbart ToonTalk så kanske problemet försvinner.

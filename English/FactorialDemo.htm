Press the Start button to run the applet. If you don't see a start button it is probably because your browser is set up so it won't run Java applets.

You need a browser that can run Java to see this. The latest browsers from Microsoft or Netscape should work fine.

This Java applet was automatically created from a team of three robots built in ToonTalk.

This program computes the factorial of a number. For example, the factorial of 5 is 5x4x3x2x1. When you start it you will be asked for a number. Just type any number and then press the enter key.

Note that this applet prompts you for a number (the prompt is from the nest's label) and displays some text along with the answer that comes from the bird's label.

The 32-bit Version of ToonTalk

System requirements.

Windows 95 or Windows NT 4.0 or later.

The 32-bit version requires the DirectX game extensions from Microsoft. You can get the latest version from Microsoft. Windows 98 and NT 4.0 already include DirectX.

The 32-bit version is included in the product CD-ROM and in Beta 15. It will also be included in Trial Release 1.1 when it is ready.

Running the 32-bit version.

When you start ToonTalk you will see a button labeled "Set Options". The second question you are asked is whether you want to set the default to run the 16 or 32-bit version of ToonTalk.

Only the 32-bit version can do the following:

  • Mixes sounds, so you can hear more than one thing at a time.
  • Run with sharp graphics (640×480 resolution, but it requires a 30 megabyte file that is not available over the net).
  • Accepts long file names.
  • Marty can talk! But this requires a 29 megabyte file that is not available over the net. Or you need to provide a text-to-speech engine that is compatible with Microsoft's Speech API. You can select the mode by editing "TextToSpeechMode=" in your toontalk.ini file.
  • Can import larger (greater than 64K) pictures.
  • Can use joysticks and game controllers. (But not on Windows NT 4.0, but should work on Windows 2000.)
  • Faster on high-end machines and video boards (and slower on old machines).
  • Read and write files and the serial port.

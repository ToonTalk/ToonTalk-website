The ToonTalk Publishing Tool

Advanced Users:

ToonTalk 3 includes a publishing tool called TT_FTP.EXE. It can be enabled by setting RunAfterSaving to TT_FTP.EXE. It can be customized by editing tt_ftp.ini found in the ToonTalk folder under My Documents. Here is a sample version:

[HTML]

Start=English\Publish.htm

Finished=English\Finished.htm

WebSite=http://www.weblabs.org.uk/wlplone

MessageBox=English\messbox.htm

[FTP]

Directory=/wlplone/Members/*/.clipboard/

Server=weblabs.dsv.su.se

[Messages]

PleaseWait=Click OK to continue. There may be a pause while your file is copied to the website.

ExpertFTPError=An error occurred while attempting to upload the file using FTP. The error is

NonExpertFTPError=Something went wrong trying to copy your file to the web site. Sorry. Try again later.

ZipError=An error occurred while creating a zip file to upload the Java applet to the web site. Sorry.

The HTML section names dialog files for when the tool starts and finishes. The FTP section lists the directory and server. Messages can be edited (e.g. for a different language).

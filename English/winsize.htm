Most people prefer the full-screen version of ToonTalk. On a few computers it doesn't work well. ToonTalk can run in any size Window but multiples of the chosen screen resolution are the fastest. If you run in a Window size larger than your screen it will look like the full-screen mode. But it might be slower or choppier than the real full-screen mode.

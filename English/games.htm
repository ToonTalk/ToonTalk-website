Games, fun things, and other stuff built in ToonTalk

Here is a list of places where you can find games and other fun things built in ToonTalk and bring them into ToonTalk to play with, take apart, modify, extend, or whatever you want.

  • Many games are available in ToonTalk 3 after clicking on the 'Visit Playground' button after starting ToonTalk.

  • The Game Place. This is the Playground research project page where games made by children and researchers are available. Warning: some of these games use lots of custom pictures and sounds and so they may take a long time to load and start the first time you try them. And some of these games are complex and might run slow on a low-end computer.

  • Playground elements and tools. Here you'll find lots of tools and pieces of games that you can use to create your own games.

  • WebLabs games. This project is mostly about modeling scientific processes but there are some ToonTalk games here as well.

  • Other ToonTalk programs. A range of programs made by us and ToonTalk users. Contribute what you've made here.

If you have a web site with ToonTalk creations on it please let us know so we can add a link here.

ToonTalk can temporarily freeze when time travel is on

If you are reading this from a local file and wish to see the latest version of the web click here.

Time travel in ToonTalk is a great way to undo things, to see things over again, to show others what you have done and more. But depending upon how many things are in your city and how fast a computer you are using it can cause ToonTalk to repeatedly freeze after a certain number of seconds. If it is freezing often we recommend you save your city, leave ToonTalk, and start again with time travel turned off.

How to Buy ToonTalk 3 

ToonTalk is available from the following sources depending upon which country it is being shipped to.

Sweden.
ToonTalk is sold by Ekelunds Förlag AB. This version has Swedish text and speech.
Portugal.
The text and speech in ToonTalk has been translated to European Portuguese and is sold by Cnotinfor of Coimbra, Portugal.
Europe (except for Sweden and Portugal), Africa, Australia, New Zealand, Hong Kong, Singapore, India, and Malaysia.
ToonTalk 3 is sold by Logotron. This version has British English text and speech.
Brazil.
Divertire, a subsidiary of FuturArte Editora, a distributor and publisher of educational and reference software in Brazil has translated all the text and speech to Brazilian Portuguese. The product is now on sale in Brazil from Divertire.
Japan and Korea.
The Japanese version is now available. Our Japanese partners have exclusive rights to sales in Korea as well as Japan.
North America and those territories not listed above.
ToonTalk 3 is sold by Animated Programs, the developer of ToonTalk. This version has American English text and speech, and a manual in the form of 33 trading cards.

 

System requirements. Windows 98, Windows ME, Windows 2000, Windows XP, or later. ToonTalk 3 runs at a reasonable speed on all Pentium computers made since 1998. A CD-ROM drive is required for installation.

Running on an Apple Macintosh. ToonTalk 3 has been tested on an Apple Mac running Virtual PC. Aside from a problem reading the position of the mouse in VirtualPC prior to version 4.0, everything works. (Note for older versions of VirtualPC that you should deselect the Virtual PC option called "PC Cursor Tracks Mac's Mouse" to get the mouse to work properly.)  If you encounter this problem, you'll need to use the arrow keys on a Mac with a PC emulator.

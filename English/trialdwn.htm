Beginning August 17, 2007 a new enhanced trial version (Internal Version 3.188) of ToonTalk is available.

The US English ToonTalk Trial program is available from servers world-wide via Simtel. If you have difficulties at Simtel it is also available here. If downloading problems persist please send email to support@toontalk.com. The installation program is named TTTrial3US.exe. Just run it once it has been downloaded. You may delete TTTrial3US.exe after running it.

In order to fit into about 30 megabytes, the trial version loads additional functionality (narrated demos and the Playground files) from the ToonTalk website as needed. The trial version expires 7 days after installation. It only includes the first of five sets of puzzles but is otherwise fully functional.

To try the UK English version visit www.logo.com.

Time permitting the following will also be included:

  • Enhanced long-distance birds. Long-distance birds are useful in version 2 but there are several limitations. In version 3 we expect birds to be able to find nests that have been moved or copied to other machines. Also a bird will be able to deliver messages even if her nest is not currently loaded into a ToonTalk.
  • MP3 playback. ToonTalk currently can playback Windows Wave files. We plan to add the ability to play MP3 files and maybe other compressed audio files.

And maybe version 3 will include some enhancements suggested by ToonTalk users. If you have suggestions, please let us know.

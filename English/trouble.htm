What to do if ToonTalk isn't working properly

If you think ToonTalk isn't working right, please click on one of the problems below. If the problem persists or isn't covered here, please report it.

  • ToonTalk is stopping or freezing.
  • ToonTalk is running too slow.
  • ToonTalk is having troubles running demos.
  • ToonTalk didn't install correctly.
  • ToonTalk seems to be having problems with Microsoft's DirectX game extensions.
  • ToonTalk is having problems using my joystick or game pad.
  • ToonTalk is having problems generating Java applets.
  • ToonTalk can't display pictures in formats other than BMP.

Click here to report a problem.

Running ToonTalk at different screen resolutions

ToonTalk can run at screen resolutions ranging from 640x480 to the highest resolution your computer is capable of (1600x1200 on some computers). The difference is that with high resolution the pictures are much sharper. The disadvantage is that it is slower and sometimes causes the animation to be jerky.  To change the resolution click on "Set Options" after starting ToonTalk. You can always change the resolution back, if you don't like the change.

You an also set the resolution to any value your computer's graphic system is capable of by using the -screen_width command line option.

If ToonTalk is running in a window (not full-screen), then these settings set the window size.

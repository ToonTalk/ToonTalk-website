History of ToonTalk Enhancements and Bug Fixes - 2007

Here is the list of bugs and enhancements in 2007. Similar lists for 1996, 1997, 1998, 1999, 2000, 2001, 2002, 2003, 2004, 2005 and 2006 are available as well.

2/18

  1. Help pages can now refer to other help pages as ../folder/name.ext while previously only full or short names were supported.

3/4

  1. A robot when no longer waiting for something to arrive on a nest sometimes caused other robots waiting on the same nest to forget that they were waiting and hence ignored the placement of something on the nest.

21/5

  1. Toontalk when launched from StartTT.exe refused to run demos whose DMO file was a DMO.URL file that specified the URL where the contents could be found. Has worked fine when toontalk.exe is launched with the appropriate command line arguments for many years.

22/5

  1. If many text-to-speech voices are installed on a system then the 'Set Options' dialog to choose a voice would not display the buttons for accepting or cancelling the option choice.

28/5

  1. If a robot was trained to vacuum a page off a notebook, it sometimes got mixed up if there was more than one notebook available.

13/6

  1. Enhanced the dialog when pausing in the puzzle game while holding something to give the option of saving what is in your hand (as ToonTalk always does when in Free Play).

18/6

  1. Changes made in June 2006 caused spurious warnings to appear in the user's log file when running old robots that used the keyboard (e.g. the Pong game), Fixed now.

6/7

  1. Trial version now enters into region-specific dialogs and warns about time remaining.
  2. Trial version now accepts a key to convert it to a product version

12/7

  1. Fixed a bug saving a city while a robot recently began running because something arrived on a nest.

24/7

  1. If ToonTalk is fetching a file from the web sometimes it repeatedly put up a dialog asking if you wanted to continue to fetch the file.

25/7

  1. Made several installer enhancements
  2. Released Beta 52 (internal version 3.188)

9/8

  1. Improved the installer so one can have multiple language versions of ToonTalk installed to different folders. Also the language of the installation version becomes the new default language (which can be reset using 'Set Options').

9/9

  1. Made further installer enhancements.
  2. Released US English product version of 3.188.

22/9

  1. Changed the expiration time of activation keys from 24 hours to 90 days.
  2. If a ToonTalk installation was copied to a new machine without running the installer it the copy would have a 75 day trial period. Reduced to 7 days now.
  3. Released 3.189 US English version.

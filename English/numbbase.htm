ToonTalk numbers can be viewed in many different bases

ToonTalk numbers can be seen or typed in any base between 2 and 36. If you have a number pad with the desired base and then type '#' to it, you'll have 0 in that base. If you drop a number on it, the sum will be shown in the new base. For example, if you have 2 and then type '#' you'll get 2#0. If you drop 5 on that it'll become 2#101 since 5 is 101 in binary.

For bases over 10, the additional digits begin with 'a' for 10 and going to 'z' for 35. So, for example, if you type "36#toon" and drop it on "36#talk" you'll get "36#1mza7". And if you drop "36#toon0000" on "36#talk" you'll get "36#toontalk". Then if you drop "36#toontalk" on 0 you'll get 2326294721000.

What are Variables in ToonTalk?

What role do variables play in programming languages?

  1. Provides a connection between multiple references to the same computational construct.

  2. Provides a means for the writers of programs to convey some computer-ignorable information to the readers of programs.

Many varieties: parameters, local variables, global variables, instance variables, class variables, ...

Consider the ToonTalk equivalent of:

temp = x;

x = y;

y = temp;

Dealing with full-screen problems in 16-bit ToonTalk

The 16-bit version of ToonTalk is no longer supported. Unless you must run on Windows 3.1, please use the 32-bit version.

Or you can try running ToonTalk is a window instead by clicking on "Set Options" after starting up ToonTalk.

Die US-englische version der ToonTalk Demoversion ist mittlerweile auf vielen Servern verfügbar. Die britisch-englische Version ist bei www.logo.com erhältlich. Falls Sie Probleme beim Herunterladen von einem Server haben, versuchen Sie es bitte zunächst bei einem anderen. Falls Sie dann immer noch Probleme haben, senden Sie bitte Email an support@toontalk.com. Das Installationsprogramm heißt ttus104.exe und sollte genau 7.656.243 Bytes  groß sein. Sie brauchen es nur nach dem Herunterladen auszuführen.

Herunterladen von ttus104.exe bei Simtel - eine Download-Bibliothek, die auf vielen Servern und verschiedenen Ländern kopiert wurde. Wenn Sie auf der Seite sind, wählen Sie bitte Ihr Land und durchsuchen das Win 3.x Archiv (da die Demoversion eine 16-Bit Anwendung ist) nach ToonTalk.

Oder versuchen Sie es bei ZDNet.

Oder versuchen Sie es mit ttus104.exe von einem Server in Maryland, USA.

Oder versuchen Sie es mit ttus104.exe von einem Server in Massachusetts, USA.

Da die Demoversion nur 7 MB groß ist, unterstützt sie nicht so viele Funktionen wie die komplette CD-ROM Version wie  z. B. bessere Grafik, Text-nach-Sprache, Joystick, Generierung von Java Applets und das Sichern von Programmen im Notizbuch. Die Lizenenz der Demoversion läuft nach 30 Tagen ab.

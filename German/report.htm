Wie man Probleme an den ToonTalk Support weiterleitet

Bei Problemen schicken Sie am besten Email an support@toontalk.com.

Bitte machen Sie folgende Angaben:

  1. Ihr Name und ihre Email-Adresse.
  2. Beschreiben Sie so gut wie möglich das Problem.
  3. Welche Version von ToonTalk Sie benutzen und ob Sie die 16-Bit- oder 32-Bit-Version verwenden.
  4. Falls die Datei dxinfo.exe auf Ihrem Computer existiert, starten Sie bitte das Programm und fügen eine Kopie der Information, die Sie sehen bei.  Dxinfo.exe sollte im Verzeichnis "\Program Files\DirectX\setup" existieren. Beschreiben Sie sonst Ihren Computer und ihr Betriebssystem so gut Sie es können.
  5. Falls möglich, beschreiben Sie wie man den Fehler wieder erzeugen kann.
Vielen Dank.

Weiter Informationen können Sie auf den Seiten des  Techchnischen Supports oder bei den  Frequently Asked Questions erhalten.

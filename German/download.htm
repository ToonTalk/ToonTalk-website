Das ToonTalk Beta 15 Release ist seit 2. Mai 1999 verfügbar.

Ein neues Beta-Release wird noch vor 30 Juni fertig sein. Bitte
warten Sie bis dahin. Bitte schicken Sie eine EßMail, wenn Sie dann benachrichtigt werden wollen.

Beta 15 ist nur als 9 MB selbstentpackenden Zip-Datei mit dem Namen ttbeat15.exe verfügbar. Es ist mit Passwort geschützt -- Senden Sie Email, um das Passwort anzufordern. Die ToonTalk Test-Version benötigt kein Passwort.
 

Zur Installation. Kopieren Sie die Datei (oder Dateien) in ein temporäres Verzeichnis (z.B. \windows\temp). Starten Sie dann bitte beta15. Sie werden dann nach einem Passwort gefragt, das Sie erhalten, wenn Sie eine E-mail schicken.

Es gibt auch eine Datei readme.txt, die Sie zuerst lesen können.

Die Lizenz der Beta-Version 15 läuft 60 Tage nach Installation ab.

Fehlerbehebung bei der Installation.
Falls Sie Fehler bei der Installation bemerken, kann es sein, daß die Datei nicht vollständig auf Ihre Festplatte kopiert wurde. Die Datei beta15.exe sollte exakt 9.527.242 Bztes lang sein. Falls dies nicht der Fall ist, versuchen Sie das Kopieren bitte nocheinmal.

Es gibt eine 32-bit ToonTalk Version, die unter Windows 95 und Windows NT 4.0 läuft. Sie wird mit der 16-Bit Version installiert, wenn Sie die Beta-Version installieren.

Diese Installation enthällt keine Erzählung zur Einführungstour. Wenn Sie Erzählungen zu Beispielen hören wollen, klicken Sie bitte hier, um nähere Informationen zu erhalten.

ToonTalk gibt es auch in Swedisch und Portugisisch.

Der Installation fehlen auch 2 Dateien, die nur für Windows NT 3.51 benötigt werden.

Klicken Sie hier, um Programme zu sehen, die nicht im Release enthalten sind.

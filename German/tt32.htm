Die 32-Bit Version von ToonTalk

Systemvoraussetzungen.

Windows 95 oder Windows NT 4.0 oder neuer. Sie müssen zuerst die 16-Bit Version installieren.

Vom Netz Installieren.

Einfach beta8c32.exe ausführen. Die 32-Bit Version benötigt die DirectX Spieleerweiterung von Microsoft. Sie können die neueste Version von Microsoft bekommen. Windows 98 und NT 4.0 enthalten schon DirectX.

Ausführen der 32-Bit Version.

 Beta8c32 hat einen Eintrag im Start -- Programme -- ToonTalk Menü, um die 32-Bit Version auszuführen. Es gibt auch die Möglichkeit, sie als Voreinstellung zu wählen.

Unterschiede zwischen der 16-Bit und der 32-bit Version.

  • Die 32-bit Version verfügt über integrierten Sound.
  • Die 32-bit Version kann als 640x480 laufen (es benötigt aber eine 30 MB Datei, die nicht über Internet verfügbar ist).
  • Akzeptiert lange Dateinamen.
  • Marty kann sprechen! Du brauchst aber einen Text-nach-Sprache Konverter, der kompatibel mit Microsofts Sprach-API ist. Du kannst dann den Modus auswählen indem Du "TextToSpeechMode=" in der toontalk.ini Datei änderst.
  • Kann größere Bilder importieren (größer als 64K).
  • Nur mit der 32-bit Version kann man Joysticks und Game Controller benutzen. (Außer bei Windows NT 4.0.)
  • Schneller auf High-end Computern und Video Boards (und langsamer auf alten Maschinen).

ToonTalk(TM) und Java(TM)

ToonTalk kann ein Java Applet für ein ToonTalk Programm erzeugen.

Um etwas in ein Java Applet zu übersetzen, nimm in ToonTalk etwas in die Hand, drücke Pause und klicke danach auf "später zurückkkehren". Du hast damit etwas aus ToonTalk exportiert, was sich nun in Windows befindet. Falls kein Fehler passiert ist, wird jetzt automatisch ein Browser gestartet und Du siehst das Programm ablaufen. Um das selbst einmal auszuprobieren, gehe auf Seite 6 zum Beispielenotizbuchs und nimm den Roboter von Seite 1, der nur immer 1 dazuaddiert. Drücke dann auf Pause und "später wiederkehren". Klicke auf "Start" sobald das Applet in Deinem Browser erscheint und schaue zu wie die Zahl größer wird. Es wird nur ein Java Applet generiert wenn man einen Roboter, einen Lastwagen oder ein Bild hält. Das Programm eines Roboters startet dann genau mit dem Behälter mit dem er trainiert wurde. Nachdem ein Lastwagen beladen wurde muß Du F8 drücken, um ihn am Abfahren zu hintern. Du kannst auch ein Textfeld in den Lastwagen legen, um dem Applet einen Namen zu geben. Wenn Du willst, daß der Roboter ein bestimmtes Notizbuch benutzt, mußt Du eine Kopie des Notizbuchs in den Lastwagen legen. Um einem Applet, das aus einem Bild (typischerweise mit einem Roboter auf der Rückseite) generiert wurde, mußt Du ein Textfeld auf die Rückseite des Bildes legen.

Diese Funktionalität benötigt einen Java-fähigen Browser wie Microsoft Internet Explorer oder Netscape Navigator. Der Compiler für Java nach Byte-Code läuft nicht unter Windows 3.1. Deshalb mußt Du das generierte Java hier irgendwie übersetzen. (Du kann hierzu den kostenlosen Compilation Service im Web verwenden.)

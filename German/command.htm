ToonTalk Kommandozeilenoptionen

Wenn Du ToonTalk über startest, wirst Du zuerst nach Deinem Namen gefragt. Alternativ kannst Du bei ToonTalk Deinen Namen auch in der Kommandozeile eingeben. Fast alle Optionen, die im folgenden beschrieben werden, können aber auch durch das Optionsnotizbuch, das Du in ToonTalk auf Seite 10 Deines Hauptnotizbuches findest, gesetzt werden.

NUR FÜR FORTGESCHRITTENE BENUTZER:

Du kannst die ToonTalk Startkonfiguration aber auch einstellen, wenn Du das Windows Programm Item (Windows 3.1) oder die Verknüpfung (Windows 95) mit dem Namen "Start ToonTalk" editierst (Alt-Enter bei selektiertem Symbol.) Nach dem Teil der TT16.exe (oder TT32.exe falls Du die 32-Bit Version ausführst) aufruft kannst Du jede der folgenden Optionen abgetrennt durch Leerzeichen anfügen. Die Voreinstellungen sollen aber ausreichen.
  • -n <Name>
    Ein Name kann von beliebiger Länge sein und kann Leerzeichen und Sonderzeichen beinhalten. Die ersten 8 Buchstaben des Namens werden als Dateiname für die Datei verwendet in der das Programm gespeichert wird. Die Voreinstellung für den Namen ist "Kein Name".
  • -h <kopf>
    Der Kopf kann 0 (Kahl), 1 (Haar) oder 2 (Helm) sein. Voreinstellung ist 2.
  • -s <Sound>
    Der Wert 0 stellt den Ton ab. Falls das System abstürzt, kann das unter Umständen an dem Zusammenspiel zwischen Soundtreiber und ToonTalk liegen. Es kann sein, daß das Setzen auf 0 das Problem dann löst. Das Problem sollte dann auch dem Hersteller mitgeteilt werden.
  • -f <Vollbild>
    Volllbilddarstellung kann 0 sein (320x240 kleiner Bildschirm), 1 (Vollbild mit Fenster-Grafik Shutdown) oder 2 (640x480, normales Fenster). 1 sieht meistens ein bißsschen ansprechender aus und ist auf einigen Computern auch ein bißchen schneller. Auf einigen Computern hängt dann allerdings die Anwendung wenn ein Fehler auftritt. Falls das der Falls ist, muß der Computer leider neu gestartet werden. 1 funktioniert auch trotz niedriger Bildschirmauflösung und Anzahl der Farben. 1 funtioniert nicht unter Windows NT. Mit 1 funktionieren die Windows <alt>-Kommandos nicht.
  • -o <Name Ausgabe Log-Datei>
    Es wird alles in der Datei abgespeichert, was mit ToonTalk gemacht wird. Mit -i kann neu die Aufzeichnung wiederholt werden. Hiermit kann nach einem Fehler gesucht werden. An die gegebene Datei wird die Erweiterung .DMO angehängt.
  • -i <Name Eingabe Log-Datei>
    Wiederholt alle aufgezeichneten Aktionen, wenn die Option -o verwendet wird. -I macht dasselbe. Es wird dann allerdings die originale Aufzeichnungsgeschwindigkeit verlangsamt, wenn es auf einem schnelleren Computer ausgeführt wird.
  • -c <Stadtgröße>
    Die Voreinstellung der Größe der Stadt ist 10x10 Blöcke (jeder Block entspricht 4 Häusern). Die größte Stadt ist 255x255. Wenn eine negative Zahl eingegeben wird, wird ToonTalk mit einer Zufallsbelegung von Häusern gestartet.
  • -b <0, 1, 2 oder 3>.
    Beschreibt wie viele Maustasten zur Verfügung stehen. Voreinstellung ist 1. Siehe auch     Maustasten für weitere Details.
  • -r <prozentuale Lesegeschwindigkeit>
    Die Lesegeschwindigkeit bestimmt wie lange ein Text in der Sprechblase verweilt. Voreinstellung ist 100 Prozent. Größere Zahlen führen dazu, daß der Text schneller wechselt.
  • -l <Schriftgröße der Sprechblase>
    Voreinstellung ist 100 Prozent.
  • -e <Schrifttyp>
    Schrift, die von Marty benutzt wird. Wenn der Schrifttyp ein Leerzeichen enthält, wie z.B. bei New Modern, wird er zu einem Wort z.B. new_modern zusammengefügt.
  • -m <als Cache benutzter Speicher>
    Die Voreinstellung ist 100 (verwendet 25% des freien Speichers). Große Zahlen beschleunigen die Anwendung etwas, laßen aber weniger Speicher für andere Dinge.
  • -u <zur Verfügung gestelltes Benutzerprofil>
    1 bedeutet, daß die Operationen, die ein Benutzer durchgeführt hat, in einer Datei mir der Erweiterung .usr aufgezeichnet und die Hilfe entsprechend angepaßt wird. 0 stellt die Aufzeichnung ab. 1 ist Voreinstellung.
  • -a <Anweisungsstufe>
    Von 0 bis 5. Bei 0 erzählt Marty alles, was er weiß. Bei 5 sagt er nichts. 2 ist die Voreinstellung.
  • -t <Vorläufiger Benutzer>
    1 bedeutet, daß keine Benutzerdatei für Notizbücher gelesen oder generiert wird. 0 ist die Voreinstellung.
  • -q <ToonTalk Geschwindigkeit>
    100 ist die Voreinstellung. Größere Zahlen beschleunigen die Animation.
  • -y <Maximale Bildwechselrate in Millisekunden>
    0 ist die Voreinstellung. 20 bedeutet z.B. eine Bildwechselrate von maximal 20 Bildern pro Sekunde - auch auf sehr schnellen Computern.
  • -k <Debuggerstufe>
    Option zur Fehlersuche. Bei 1 erzählt Marty mehr. Die Datei <BENUTZER>.log enthält dann mehr Informationen über aufgetretene Probleme. Die Voreinstellung ist 0.
  • -bird <von außerhalb Vögel zurückkehren sehen>
    Bei 0 brauchen Vögel keine Zeit, um nach dem Transport von Gegenständen zu ihrem Nest zurückzukehren, wenn der Programmierer von außsserhalb eines Hauses zuschaut. Die Voreinstellung ist 1.
  • -j <führt die japanische Version von Windows 3.1 aus>
    1 bedeutet, daß kein Titel angezeigt wird. Dies scheint eine Inkompatibilität zwischen Win 3.1J und WinG von Microsoft zu beheben: Beim ersten Mal wird kein Text angezeigt.
  • -puzzle <Dateiname>
    Started ToonTalk mit einem Rätsel durch den gegebenen Dateinamen.
  • -next_puzzle
    Started ToonTalk mit dem nächsten Rätsel bezüglich eines bestimmten Benutzernamens. Es startet mit dem ersten Rätsel, wenn noch kein Rätsel gelöst wurde.
  • -autodemo <0, 1 oder 2>.
    0 ist die Voreinstellung und schaltet den automatischen Demomodus aus. 1 bedeutet, daß, wenn 30 Sekunden nichts passiert, das System immer wieder die Einführungstour zeigt. Im Zustand "Abenteuer spielen" oder "freies Spiel" wartet das System 120 Sekunden. Dies wird in einer Zeile in der Datei toontalk.ini im Windows-Verzeichnis eingestellt: "AutoDemoMaxIdle=120". Bei der Option -autodemo 2 werden alle Beispiele durchlaufen werden, wenn nichts passiert.
  • -subtitles <Geschwindigkeit>
    Falls bei einer Demonstration Untertitel gezeigt werden, dann werden sie mit einer Lesegeschwindigkeit, deren Voreinstellung 100 ist, gezeigt. Wenn die Geschwindigkeit 0 ist, werden keine Untertitel gezeigt. Die Sprache für die Untertitel wird durch "SubtitlesSuffix=ust" im Abschnitt "Executables" von toontalk.ini eingestellt. Wobei ,,ust'' das Suffix für amerikanische und englische Untertitel und ,,set'' das Suffix für schwedische Untertitel ist.
  • -speak <0, 1> 1 schaltet den Text nach Sprache Konverter an, falls die 32-Bit Version benutzt wird. 0 schaltet ihn ab. 1 ist die Voreinstellung der 32-Bit Version und 0 die der 16-Bit Version.
  • -balloon <0,1> 1 schaltet Martys Sprechblase an. Default is 1.
  • -tts_uses_ds <0,1> Bei 1 benutzt der Text-nach-Sprache Konverter DirectSound. Die Option erlaubt simultane Sprache und Soundeffekte. Die Voreinstellung ist 0, da auf vielen Systemen dies sehr lange Verzögerungen und Hintergrundzischen verursachen kann. Diese Option ist nur für die 32-Bit Version. Probier es aus, wie es sich auf Deinem System anhört.

Anpassen der ToonTalk Installation

Nur für vorgschrittene Benutzer:

Sie k”nnen die Datei toontalk.ini editieren, die Sie im Windows Verzeichnis finden.

  • 32BitDefault. Wenn der Wert 0 ist, wird die 16-Bit Version von ToonTalk gestartet. Wenn der Wert 1 ist, wird die 32-Bit Version gestartet. Der Wert kann durch -32 <0,1> in der Kommandozeile überschrieben werden.
  • GenerateLogs. Dieser Eintrag erzeugt Dateien, die wiederabgespielt werden können, mit dem Namen log<Nummer>.dmo im ToonTalkverzeichnis. Eignet sich, um Probleme reproduzieren zu können. Der Wert entspricht der Anzahl an Logs, die gespeichert werden.
  • AutoDemoMaxIdle. Der Wert entspricht der Zeit in Sekunden, die gewartet wird bevor mit -autodemo 1 oder 2 gestartete Demos neu gestartet werden.
  • ShowSubtitles. Falls der Wert 0 ist zeigt ToonTalk keine Subtitel bei Demos.
  • NarrationFileSuffix. Der Wert muß zur Zeit SPK für englische und TAL für schwedische Erzählungen sein. Dies ist die Voreinstellung für die -language Option.
  • SubtitlesSuffix. Der Wert muß zur Zeit entweder UST für englische oder SET für schwedische Subtitel sein. Subtitel können in einer anderen Sprache als die Erzählung sein.
  • TextToSpeechMode. Dieser Eintrag ist nur relevant, wenn Sie einen Text-nach-Sprache Konverter installiert haben und diesen den in der 32-Bit ToonTalk Version befindlichen Konverter vorziehen.
  • JoystickDeadZone. Dieser Eintrag ist nur relevant, wenn sie die 32-Bit Version von ToonTalk ausführen und einen Joystick oder Game Controller angeschlossen haben. Ein Joystick zeigt manchmal eine Bewegung an, ohne daß man ihn berührt hat. Der Eintrag sagt ToonTalk, welche kleineren Bewegungen noch ignoriert werden sollen. Der Wert kann von 0 bis 10.000 gehen. Die Voreinstellung ist 2.500.
  • [ButtonKeyboardEquivalents]. Dies erlaubt die Funktion von Joysticktaste 3 und höher einzustellen (Joysticktaste 3 entspricht der 3. Maustaste.). Es können Spezialtasten definiert werden indem sie in "<>" eingeschlossen werden. Die verfügbaren Spezialtasten sind <F1>, <F2>, ... <F12>, <Escape> und <Leertaste>. Die Voreinstellungen sind zum Beispiel
    • 3='-'
    • 4=<F1>

How to Install ToonTalk Extensions

For advanced users only:

ToonTalk extensions are not installed by default. One reason for this is that they enable robots to changes things outside of ToonTalk's safe world. The file extension, for example, enables ToonTalk robots to read and change (or destroy) any file on your computer. This power can be used for good or mischief. If you install this extension, then be sure you trust people who give you ToonTalk programs to run.

Extensions currently only work in the 32-bit version of ToonTalk. If you are interested in running extensions in the 16-bit version send email to support@toontalk.com.

An extension is installed if the DLL file starting with "TT" followed by the extension name is in your ToonTalk directory. For example, the file extension file is "TTFile.dll".

If you are installing from a CDROM, then choose the "Custom Installation" option. Select "Program Files" and click on the "change" button. You should then remove check marks from every component except the ones that are labeled with the extensions you desire. Alternatively copy the appropriate files from the \toontalk\extend subdirectory.

If you are installing over the Internet then download and copy TTFile.dll to your ToonTalk directory to install the file extension.

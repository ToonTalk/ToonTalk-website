ToonTalk can speak many languages!

Well, just 3 languages right now:

  1. English (US and UK). The US version is included in the current beta test release.
  2. Swedish. This includes translations of dialogs, everything Marty says, the puzzles, and the documentation. To install just run swedish.exe in Windows. To run ToonTalk in Swedish see below. The ToonTalk CDROM also includes Swedish narration for most of the demos. The CDROM also enables Marty to speak in Swedish. The translation was done by Lennart Mogren and Tobias Zetterlund. ToonTalk is for sale in Sweden.
  3. Portuguese (Brazil). This currently includes translations of dialogs and everything that Marty says. To install just run port.exe, in Windows. The translation was done by Leo Burd. NOTE -- THIS IS CURRENTLY UNDER REVISION. SEND EMAIL TO support@toontalk.com IF INTERESTED.
  4. German. In progress. Nearly all of this web site has been translated. Click here to explore the German translation.
  5. Spanish. In progress.
  6. Your language here. If you are interested in helping translate ToonTalk to some other language please let us know.

To use an installed language just add the line "-language Swedish" or "-language Portuguese" to the command line for starting ToonTalk.

If you wish to make it easy for users to switch languages then you should edit the toontalk.ini file in your Windows directory to add the line

Language2=Swedish

after the line Language1=English.

After doing so "Set Options" after starting ToonTalk will offer a choice of languages.

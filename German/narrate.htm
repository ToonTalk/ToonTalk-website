Narration for ToonTalk demos

Narrations for the demos are available on the CD-ROM. The Introductory Tour and the first act of the Ping Pong demo have narration which is also available by downloading self-extracting zip files. To install Ping Pong part 1, for example, just run pongact1.exe and extract the file to the "demos" subdirectory of your ToonTalk installation.

  • Download narration for the Introductory Tour (4M compressed; 9M unzipped)
  • Download narration for part 1 of Ping Pong demo. (4M compressed; 9M unzipped)

After you hear a demo you can delete the SPK file if you don't plan on hearing the narration again.

The other demos have narration as well, but the files are not available for downloading. These files contain the American English narrations. The British English and the Swedish versions are only available on the CDROM.

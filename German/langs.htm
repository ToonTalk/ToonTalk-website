ToonTalk spricht viele Sprachen!

Es sind wieder 3 Sprachen hinzugekommen:

  1. Englisch (US). Das umfaßt das jetzige Betatest-Release.
  2. Schwedisch. Umfaßt die Übersetzung von Dialogen, alles was Marty erzählt, die Rätsel und die Dokumentation. Zur Installation muß swedish.exe unter Windows ausgeführt werden. Die ToonTalk CD-ROM enthält auch schwedische Erzählungen der meisten Demos. Die Übersetzung wurde von Lennart Mogren und Tobias Zetterlund durchgeführt. ToonTalk kann man in Schweden kaufen.
  3. Portugiesisch (Brazilianisch). Umfaßt zur Zeit die Übersetzung der Dialoge und alles was Marty sagt. Zur Installation muß port.exe unter Windows ausgeführt werden. Die Übersetzung wurde von Leo Burd durchgeführt.
  4. Deutsch. In Arbeit. 
  5. Spanisch. In Arbeit.
  6. Andere Sprachen. Falls Sie daran interessiert sind, bei der Übersetzung in eine andere Sprache zu helfen schreiben Sie uns.
Um eine installierte Sprache zu benutzen, fügen Sie bitte "-language Swedish" oder "-language Portuguese" in der Kommandozeile beim Starten hinzu.

Falls Sie es dem Benutzer einfacher machen wollen, zwischen zwei Sprachen zu wechseln, sollten Sie die Datei toontalk.ini in Ihrem Windows Verzeichnis ändern und eine Zeile:

Language2=Swedish

nach der Zeile Language1=English hinzufügen. Hiernach können Sie die zweite Sprache unter Optionen auswählen.

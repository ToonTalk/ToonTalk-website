Kann ich ToonTalk kaufen?

ToonTalk ist über folgende Anbieter zu beziehen.

Schweden.
ToonTalk wird von Ekelunds Förlag AB vertrieben. Version wird mit schwedischem Text und Sprache geliefert.
Europa (außer Schweden), Afrika, Australien, Neuseeland, Hong Kong, Singapur, Indien und Malaysia.
ToonTalk wird von Logotron, ein Tochterunternehmen von Addison Wesley Longman, vertrieben. Version wird mit britisch-englischem Text und Sprache geliefert.
Brasilien.
Divertire, ein Tochterunternehmen von FutureArte Editora, ein Distributor und Publisher von Lern- und Referenzsoftware hat zugesagt, ToonTalk nach Portogisisch zu übersetzen und zu verkaufen.
Andere Länder.
Animated Programs, der Entwickler von ToonTalk. Version mit amerikanisch-englischem Text und Sprache und einem Handbuch in Form von 33 Spielkarten.

System Anforderungen. Windows 3.1, Windows 95, Windows 98, Windows NT, oder spätere Version auf den meisten 486 PCs und allen Pentiums. ToonTalk benötigt eine Grafikkarte mit 256 Farben und ein CD-ROM Laufwerk zur Installation. Unter Windows 3.1 sind viele Eigenschaften wie Text-nach-Sprache nicht verfügbar. Die Installation einer Soundkarte wird empfohlen. ToonTalk wurde auf MacIntosh unter SoftWindows und Virtual PC getestet. Dort gibt es zur Zeit nur Probleme mit dem Mauszeiger. Bis zur Behebung dieses Bugs müssen hier die Pfeiltasten oder ab Windows 5.0.2 oder Virtual PC 2.1 der Joystick benutzt werden.
